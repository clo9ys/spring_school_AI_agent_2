## Track: A + C

### 🏎️ Car Scout AI: Интеллектуальный авто-эксперт

**Car Scout AI** — это автономный агент на базе LLM, спроектированный для глубокого анализа автомобильного рынка. Он умеет находить лучшие предложения по бюджету, извлекать технические характеристики из глобальных справочников и давать рекомендации по надежности

###  Ключевые особенности

- **Track A (Agent + API):** Интеграция с **Wikipedia API**. Агент в реальном времени извлекает историю модели, технические параметры и интересные инженерные факты
- **Track C (Data Agent):** Работа с кастомным датасетом на **40+ популярных автомобилей**. База содержит актуальные цены, оценки надежности (0-10) и перечень типичных неисправностей
- **Умное ранжирование:** Агент использует алгоритм ценового соответствия, чтобы предлагать автомобили в зависимости от бюджета пользователя

###  Стек технологий

- **LLM:** Ollama (`qwen2.5-coder:7b`) — специально выбрана для точного вызова инструментов
- **Framework:** `smolagents` (Hugging Face) — для реализации логики `ToolCallingAgent`
- **Environment:** `uv`
- **API:** Wikipedia API
